# Как удалить ветку в git

## tl;dr

### Удалить локальную ветку:

```shell
git branch -D local-branch-name
```

### Удалить ветку в репозитории (в github, gitlab, gitea и тд):

```shell
git push origin --delete remote-branch-name
```
